Analysis Process

I used the vaderSentiment library to analyze tweet sentiment:

sentiment_analysis.py
from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer

analyzer = SentimentIntensityAnalyzer()

for tweet in tweets:
    scores = analyzer.polarity_scores(tweet)
    print(f"Compound score: {scores['compound']}")

The compound score ranges from -1 (most negative) to +1 (most positive).

Critical Reflection

üìê Classification Logic

This project reveals how algorithmic sentiment analysis imposes categorical boundaries on human emotion. VADER's compound score reduces complex cultural expressions to a single number between -1 and +1.

Question to consider: What nuances are lost when we classify sentiment as simply "positive," "negative," or "neutral"?

Topic modeling with LDA appears to "discover" themes in text, creating an illusion of machine intelligence. However, the algorithm simply identifies statistical word co-occurrences‚Äî the interpretation and meaning-making is entirely human.
